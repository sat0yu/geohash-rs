# Geohash

A bare-minimum geohash encoder

## Usage

```rust
geohash_rs::encode(56.64911, 10.40744, 8)  // => "u4p"
geohash_rs::encode(57.64911, 10.40744, 16) // => "u4pruy"
geohash_rs::encode(57.64911, 10.40744, 32) // => "u4pruydqqvj8"
```
